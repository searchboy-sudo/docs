# Introduction

A free and ethical photo sharing platform, powered by ActivityPub federation.

Pixelfed is freely licensed under the AGPL license.

# Communication

The ways you can communicate on the project are below. Before interacting, please read through the Code Of Conduct.

  • IRC: #pixelfed on irc.freenode.net (#freenode_#pixelfed:matrix.org through Matrix)
  • Project on Mastodon: @pixelfed@mastodon.social
  • E-mail: hello@pixelfed.org
